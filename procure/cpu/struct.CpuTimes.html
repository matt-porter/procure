Details about current CPU time utilization.

This object provides insight into the amount of time, measured in USER_HZ, that the system spent in various modes. These are hertz counters and not necessarily useful on their own, but against itself over some duration.

Fields

fn total() -> Result<CpuTimes>

Get a CpuTimes object filled with the total cpu times aggregated for all cores.

let times = CpuTimes::total().unwrap();

        idle: 43336958,

        guest: 0,
        guest_nice: 0,
    }
);

fn per_cpu() -> Result<Vec<CpuTimes>>

Get a Vec of CpuTimes objects, one for each core.
